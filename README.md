## 🧭 About Me

我是一名关注 **Java 后端开发、AI 检索系统、数学建模与低空工程平台设计** 的本科生，喜欢把真实场景中的复杂问题抽象为模型，并通过后端服务、数据系统、算法评测和可视化界面实现落地。

> Undergraduate developer focused on Java backend systems, AI search, RAG applications, mathematical modeling, and low-altitude engineering platforms.

* 🔭 目前关注：Spring Boot / Spring Cloud、AI Search、RAG、推荐系统、数据可视化
* 🛩️ 场景兴趣：低空经济、无人机调度、空域资源协同、时空冲突检测
* 📐 建模方向：优化建模、统计分析、评价模型、调度问题、工程决策支持
* 🚀 长期目标：把算法能力、建模思维和工程系统结合起来，构建可用、可解释、可迭代的软件平台

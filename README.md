<div align="center">

```mermaid
flowchart LR
    A[Problem<br/>问题] --> B[Assumptions<br/>假设]

    C --> D[Algorithm<br/>求解]
    D --> E[Evaluation<br/>评价]
    E --> F[System<br/>落地]
```
